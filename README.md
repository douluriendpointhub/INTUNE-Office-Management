This repository contains all day-to-day operational scripts, tools, software packages, configuration files, and standard logos used in the Office/Enterprise environment.

It is mainly designed to support Intune / SCCM/ Endpoint / Office IT operations by centralizing frequently used resources such as:

Automation scripts (PowerShell, Batch, etc.)

Application deployment files & silent install/uninstall commands

Intune related configurations & references

Commonly used software installers

Office / company standard logos & branding assets

Useful documentation and quick reference notes

Goal:
To make daily IT/Intune admin work faster, consistent, and error-free by having everything in one place, so operational tasks can be completed easily and efficiently in an office environment.
